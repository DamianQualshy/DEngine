# Drakania Engine
Drakania Engine is a collective of scripts, intended to help new scripters with making their own gamemodes for Gothic 2 Online. It doesn't require any outside modules or frameworks.

## What's Done
- RolePlay Chat
- Admin Commands
- Player Class
- Primitive Calendar
